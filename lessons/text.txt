Dot notation — это способ доступа к свойствам (и методам) объекта в JavaScript (и других языках).
Используется точка (.), чтобы обратиться к конкретному свойству объекта.

Bracket notation — это способ обращения к свойствам объекта в JavaScript,
 при котором имя свойства указывается как строка в квадратных скобках.

DOM:
DOM - another built-in object. DOCUMENT OBJECT MODEL
DOM combines JS and HTML together. It gives JS full control of the webpage.

The document object represents / models the webpage.

document.querySelector() - lets us get any element from the page and put it inside JavaScript.

innerHTML -  это свойство в JavaScript, которое используется для чтения или изменения HTML-содержимого элемента на веб-странице. Gives us control of the whole HTML section.
